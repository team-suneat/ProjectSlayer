## 📝 개발 일지

### 오늘 한 일

**1. 리소스 관리 시스템 확인 및 활용**
- 재화 관리 시스템(VCurrency)의 기능을 확인하고 활용 방법을 파악했습니다.
- 재화 획득/소비, 조회, 검증 기능과 이벤트 시스템을 확인했습니다.
- 경험치 관리 시스템(VCharacterLevel)의 기능을 확인했습니다.
- 경험치 획득, 레벨업 계산, 이벤트 시스템을 파악했습니다.

**2. 재화 표시 UI 구현**
- HUD에 재화를 상시 표시하는 UI 컴포넌트를 구현했습니다.
- 재화별 아이콘을 표시하고 수량을 숫자 포맷팅하여 보기 좋게 표시했습니다.
- 재화 변경 이벤트를 구독하여 실시간으로 UI가 업데이트되도록 연동했습니다.
- 플레이어 캐릭터 준비 시 초기 재화 값을 자동으로 설정하도록 구현했습니다.

**3. 경험치 게이지 UI 구현**
- 경험치 게이지를 표시하는 UI 컴포넌트를 구현했습니다.
- 현재 경험치와 필요 경험치를 표시하고, 레벨 정보도 함께 표시합니다.
- 경험치 획득 시 게이지가 자동으로 업데이트되도록 이벤트를 연동했습니다.
- 레벨업 시 게이지가 자동으로 리셋되도록 처리했습니다.

**4. UI 연동 및 테스트 기능 추가**
- 데이터 변경 시 UI가 자동으로 반영되도록 이벤트 기반 연동을 완료했습니다.
- 레벨업 버튼과 경험치 획득 버튼을 추가하여 기능을 테스트할 수 있도록 했습니다.
- 버튼 클릭으로 재화와 경험치를 획득할 수 있는 테스트 기능을 추가했습니다.

**5. UI 개선 및 버그 수정**
- 토글과 버튼에 클릭 효과를 추가하여 사용자 경험을 개선했습니다.
- 빠른 터치에 의한 버그를 수정하여 연속 클릭 시 발생하는 문제를 해결했습니다.
- 게임 내 UI 폰트와 텍스트 크기를 통일하여 일관된 디자인을 적용했습니다.
- 언어 변경에 따른 버그를 수정하여 다국어 지원 시 발생하는 문제를 해결했습니다.

**6. 강화 페이지 기능 구현**
- 강화 페이지의 능력치 강화 구매 기능을 구현했습니다.
- 강화 페이지의 레벨에 따라 세부 정보를 갱신합니다.

**7. 성장 페이지 기능 구현**
- 성장 페이지의 능력치 성장 구매 기능을 구현했습니다.
- 성장 페이지의 레벨에 따라 세부 정보를 갱신합니다.

### 내일 할 일
- 조건 능력치가 일정 레벨에 도달하지 않았을 때 예외 처리를 추가하여 오류를 방지합니다.
- 스탯 포인트 텍스트가 갱신되지 않는 버그를 수정합니다.
- 스탯 포인트 갱신 시 성장 레벨이 갱신되지 않는 버그를 수정합니다.
- 성장 능력치 내용이 설정되지 않는 버그를 수정합니다.
