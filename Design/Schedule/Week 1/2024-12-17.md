**2. 전투 시스템 정리**
- 사용하지 않는 기능들을 삭제하여 코드를 가볍게 만들었습니다.

**3. 체력바 시스템 개선**
- 플레이어 전용 체력바를 새로 만들었습니다.

- 코드 용어를 "Life"에서 "Health"로 통일하여 이해하기 쉽게 정리했습니다.

**4. 캐릭터 정보 팝업 UI 구현**
- 캐릭터 정보를 확인할 수 있는 팝업창의 코드를 작성했습니다.
- 무기/악세사리 슬롯과 능력치 표시 영역을 구현했습니다.
- (Unity에서 실제 UI 배치 작업은 아직 남아있습니다)

**5. 캐릭터 페이지 UI 구현**
- 강화/성장/승급 탭을 전환할 수 있는 캐릭터 페이지를 만들었습니다.
- 캐릭터 레벨과 경험치를 표시하고 레벨업 버튼을 추가했습니다.
- 강화 페이지는 기능 구현을 완료했고, 성장/승급 페이지는 기본 틀만 만들어두었습니다.
