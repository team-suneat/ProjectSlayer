### 4.9.4 재화 종류 설명

#### 등급 표기법
- **C**: 일반(Common)
- **G**: 고급(Grand)
- **R**: 레어(Rare)
- **E**: 영웅(Epic)
- **L**: 전설(Legendary)
- **M**: 신화(Mythic)
- **A**: 불멸(Immortal)

#### 속성석 종류
- **불 속성석**: 불(Fire) 속성 스킬 학습에 필요
- **물 속성석**: 물(Water) 속성 스킬 학습에 필요
- **바람 속성석**: 바람(Wind) 속성 스킬 학습에 필요
- **땅 속성석**: 땅(Earth) 속성 스킬 학습에 필요

#### 무기/악세사리 등급
- 등급 뒤의 숫자는 해당 등급 내의 레벨을 의미합니다.
- 예: R4 = 레어(Rare) 등급 4레벨 무기/악세사리
