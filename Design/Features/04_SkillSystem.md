- [ ] 스킬 슬롯 시스템

  - [ ] 레벨에 따른 스킬 슬롯 해금 (최대 10개)
  - [ ] 스킬 슬롯 해금 조건 관리
- [ ] 스킬 카드 해금 시스템
  - [ ] 일정 레벨마다 스킬 카드 해금
  - [ ] 스킬 카드 해금 조건 관리
  - [ ] 해금된 스킬 카드 목록 관리
- [ ] 스킬 카드 획득 시스템

  - [ ] 스킬 카드 보유 상태 관리
- [ ] 스킬 학습 시스템
  - [ ] 스킬 카드 획득 후 스킬 배우기
  - [ ] 스킬 학습 재료 시스템
    - [ ] 각 스킬마다 정해진 재료 요구사항
    - [ ] 스킬 카드만 필요한 경우
    - [ ] 스킬 카드 + 일정 등급 이상의 특정 무기 또는 악세사리가 필요한 경우
    - [ ] 재료 조건 체크 시스템
    - [ ] 재료 소비 처리
  - [ ] 배운 스킬 목록 관리
  - [ ] 스킬 학습 조건 체크
- [ ] 스킬 할당 시스템
  - [ ] 배운 스킬을 해금된 스킬 슬롯에 할당
  - [ ] 스킬 슬롯에 스킬 장착/해제
  - [ ] 스킬 할당 UI
