# 동료 시스템

## 6.1 동료 기본 시스템
- [ ] 동료 획득 시스템
  - [ ] 동료 해금 조건 관리
  - [ ] 동료 해금 상태 저장
  - [ ] 동료 목록 관리
- [ ] 동료 정보 표시
  - [ ] 동료 기본 정보 (이름, 클래스, 속성, 종족)
  - [ ] 동료 해금 상태 표시
- [ ] 동료 종류 (총 4명)
  - [ ] 엘리 (궁수, 바람 속성, 엘프 종족)
    - [ ] 해금 조건: 시작의 숲 클리어
  - [ ] 지크 (전사, 땅 속성, 인간 종족)
    - [ ] 해금 조건: 탐욕의 언덕 클리어
  - [ ] 미호 (도적, 불 속성, 인간 종족)
    - [ ] 해금 조건: 그림자 계곡 클리어
  - [ ] 루나 (마법사, 물 속성, 엘프 종족)
    - [ ] 해금 조건: 홉고블린 족장 처치

## 6.3 동료 강화패시브 시스템
- [ ] 강화패시브 기본 시스템
  - [ ] 강화패시브 레벨 관리
  - [ ] 강화패시브 효과 적용
- [ ] 강화패시브 카테고리 1 (기본 패시브)
  - [ ] 각 패시브 최대 레벨: 100
  - [ ] 엘리 패시브: 집중 사격, 숲의 도움, 요정의 노래
  - [ ] 지크 패시브: 투지, 광기, 검무
  - [ ] 미호 패시브: 섀도우 스텝, 섀도우 댄스, 골드 러시
  - [ ] 루나 패시브: 마나 도프, 마나 증폭, 전쟁의 지혜
- [ ] 강화패시브 카테고리 2 (해금 필요)
  - [ ] 각 패시브 최대 레벨: 100
  - [ ] 해금 조건: 동료 레벨 기반 (n-2-1, n-2-2, n-2-3)
  - [ ] 엘리 패시브: 바람의 노래, 요정의 찬가, 바람의 이해
  - [ ] 지크 패시브: 소울 캐치, 연성의 지혜, 대지의 이해
  - [ ] 미호 패시브: 붉은 탐욕, 골드러시 II, 화염의 이해
  - [ ] 루나 패시브: 심해의 노래, 심연의 찬가, 바다의 이해
- [ ] 강화패시브 카테고리 3 (고급 패시브)
  - [ ] 속성 데미지 패시브: 최대 레벨 1500
  - [ ] 기타 패시브: 최대 레벨 100
  - [ ] 해금 조건: 동료 레벨 기반 (n-3-1, n-3-2, n-3-3)
  - [ ] 엘리 패시브: 요정의 손길, 약점 간파, 바람의 교감
  - [ ] 지크 패시브: 무질서, 봉쇄, 대지의 교감
  - [ ] 미호 패시브: 캐스팅, 그림자 낙인, 화염의 교감
  - [ ] 루나 패시브: 룬 마법, 마법 차단, 바다의 교감
- [ ] 강화패시브 레벨업 시스템
  - [ ] 패시브 강화 시 레벨 증가 (1/n씩 증가)
  - [ ] 레벨 증가에 따른 속성 스킬 데미지 증가
  - [ ] 최대 레벨 제한 체크

## 6.4 동료 전직 시스템
- [ ] 전직 기본 시스템
  - [ ] 전직 레벨 관리 (0 = 전직 안함, 1~7 = 1차~7차)
  - [ ] 전직 조건: 전직 대상과 1대1 전투 승리
  - [ ] 전직 보상: 외형 변경, 다이아몬드, 에메랄드 지급
- [ ] 전직 제약 시스템
  - [ ] 1~2차 전직: 대상과 같은 속성 스킬 사용 불가
  - [ ] 3~4차 전직: 같은 속성 스킬 사용 시 대상 체력 증가 디버프
  - [ ] 5~6차 전직: 같은 속성 스킬 사용 시 체력 증가 + 전투 시간 감소 디버프
  - [ ] 7차 전직: 같은 속성 스킬 사용 시 체력 증가 + 전투 시간 감소 + 최대 마나 감소 디버프
  - [ ] 디버프 효과는 사용한 동일 속성 스킬 개수에 비례하여 증가
- [ ] 전직 해금 시스템
  - [ ] 각 전직의 7번째 승급 후 다음 전직 해금
  - [ ] 2차, 3차, 4차, 5차, 6차, 7차 전직 순서대로 해금

## 6.5 동료 승급 옵션 시스템
- [ ] 승급 옵션 기본 시스템
  - [ ] 전직마다 승급 옵션 해금
  - [ ] 옵션 개수: 9개 (0~8 인덱스)
  - [ ] 옵션 잠금/해제 시스템
  - [ ] 잠금 해제 시 변경 가격 증가
- [ ] 승급 옵션 등급 시스템
  - [ ] 옵션 등급 확률
    - [ ] 회색: 23%
    - [ ] 초록: 20%
    - [ ] 주황: 30%
    - [ ] 보라: 20%
    - [ ] 빨강: 5%
    - [ ] 민트: 2%
  - [ ] 옵션 타입별 수치 범위 관리
    - [ ] 추가 공격력: 3%~20%
    - [ ] 치명타데미지: 5%~35%
    - [ ] 추가 체력: 5%~35%
    - [ ] 추가 체력 회복량: 5%~35%
    - [ ] 추가마나: 3%~15%
    - [ ] 추가 마나 회복량: 3%~15%
    - [ ] 몬스터 처치 골드: 3%~20%
    - [ ] 명중: 3%~15%
    - [ ] 회피: 3%~15%
    - [ ] 추가 경험치: 1%~8%
    - [ ] 상태 이상 저항: 3%~20%
- [ ] 승급 옵션 관리 기능
  - [ ] 3차 전직 오픈 시 4만 다이스로 일괄 변경 기능
  - [ ] 옵션 변경 비용 계산
  - [ ] 옵션 효과 적용
