# 모험 시스템

## 7.1 모험 구조
- [ ] 지역 시스템
  - [ ] 지역 8개 구성
  - [ ] 지역 선택 및 이동
  - [ ] 지역 정보 표시
- [ ] 스테이지 시스템
  - [ ] 각 지역당 스테이지 20개
  - [ ] 스테이지 진행 시스템
  - [ ] 스테이지 클리어 조건
- [ ] 웨이브 시스템
  - [ ] 각 스테이지당 웨이브 20개
  - [ ] 웨이브 진행 관리
  - [ ] 웨이브 클리어 조건
  - [ ] 20웨이브 도달 시 상자 몬스터 스폰
  - [ ] 상자 몬스터 처치 후 웨이브 리셋 (1웨이브부터 재시작)
- [ ] 몬스터 시스템
  - [ ] 웨이브별 몬스터 종류 설정
    - [ ] 예: 1지역 1스테이지 - 버섯 몬스터
    - [ ] 예: 1지역 2스테이지 - 버섯 또는 슬라임 몬스터
  - [ ] 웨이브별 몬스터 수량 설정
    - [ ] 예: 1지역 1스테이지 - 각 웨이브당 1마리
    - [ ] 예: 1지역 2스테이지 - 각 웨이브당 4마리
  - [ ] 몬스터 스폰 관리
  - [ ] 몬스터 타입별 설정 (버섯, 슬라임 등)
  - [ ] 상자 몬스터 시스템
    - [ ] 20웨이브 도달 시 상자 몬스터 등장
    - [ ] 상자 몬스터 처치 시 보상 (더 많은 골드, 경험치)
    - [ ] 상자 몬스터 처치 후 웨이브 리셋 처리

## 7.2 드랍 시스템
- [ ] 스테이지별 드랍 확률 시스템
  - [ ] 스테이지마다 정해진 드랍 확률 설정
  - [ ] 예: 1지역 1스테이지 - 일반 4등급 무기 0.1% 드랍
  - [ ] 예: 1지역 2스테이지 - 일반 4등급 무기 0.2% 드랍
- [ ] 드랍 아이템 종류 관리
  - [ ] 무기 드랍 시스템
  - [ ] 장비 등급 시스템 (일반 4등급 등)
  - [ ] 드랍 확률 계산 및 적용
- [ ] 드랍 아이템 획득 처리
