## 1. 무기 시스템

### 1.1 무기 기본 효과
무기는 다음 스탯을 증가시킨다:
- **데미지**
- **치명타 데미지**
- **골드 획득량**

#### 등급별 효과 범위
| 등급 범위 | 데미지 | 치명타 데미지 | 골드 획득량 |
|-----------|:------:|:-------------:|:-----------:|
| 일반 ~ 고급 | ✓ | - | - |
| 레어 ~ 전설 | ✓ | ✓ | - |
| 신화 ~ 불멸 | ✓ | ✓ | ✓ |

### 1.2 무기 강화
- 기본 최대 강화 레벨: **200LV**
- 같은 장비 **5개 융합** → 다음 등급 장비 획득
- 최종 무기 **"오르"**는 뽑기로 획득 불가
  - 신화 1등급 무기 5개 융합으로만 획득 가능

### 1.3 무기 각성
> 마수사냥 스킬 습득 후 각성 시도 가능

#### 각성 진행 방식
| 단계 | 재료 | 결과 |
|------|------|------|
| 일반 각성 (5회) | 신화 1등급 무기 1개 | 별 1개 획득 (최대 5개) |
| 1~3차 최종 각성 | 신화 1등급 무기 4개 | 외형 변경 + 전용 애니메이션 |
| 4차 최종 각성 | 빛의 조각 10,000개 | 최종 외형 + 전용 애니메이션 |

#### 각성 효과
- 각성 1회당 모든 무기의 **최대 레벨 +50** 상승
- 최종 각성마다 **외형 변경**
- 최종 각성마다 **전용 애니메이션** 부여

## 2. 악세사리 시스템
