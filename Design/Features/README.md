# 방치형 게임 인게임 기능 목록

이 폴더에는 방치형 게임의 인게임 기능들을 대분류별로 나누어 정리한 문서들이 포함되어 있습니다.

## 개요
타이틀 씬 - 로딩 씬 - 게임 메인 씬을 구현하고, 인게임의 메인이 되는 기능들을 구현합니다.
설정도 구현하며, 인앱 결제나 광고는 구현하지 않습니다.

## 주요 기능 분류
- **캐릭터**: 캐릭터 레벨, 스탯, 업그레이드 시스템
- **스킬**: 스킬 해금, 레벨업, 액티브/패시브 스킬
- **장비**: 장비 착용, 강화, 효과 시스템
- **동료**: 동료 획득, 성장, 전투 참여
- **모험**: 스테이지/웨이브 진행, 보상 시스템
- **상점**: 아이템 구매, 상점 관리 시스템
